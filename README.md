# 🚀 SmartStock API

API REST desenvolvida com Spring Boot para gerenciamento de estoque de produtos.

O projeto permite cadastrar, listar, atualizar e remover produtos de um estoque utilizando PostgreSQL como banco de dados.

---

# 🛠️ Tecnologias utilizadas

- Java 21+
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- Maven
---

# 📦 Funcionalidades

✅ Cadastro de produtos  
✅ Listagem de produtos  
✅ Busca de produto por ID  
✅ Atualização de estoque  
✅ Remoção de produtos  
✅ Validação de dados  
✅ Tratamento de exceções
